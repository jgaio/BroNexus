To get the project to compile properly...

1. Open the project in eclipse
2. Right click on "BroNexus" at the top of the Package Explorer, go to "Build Path" -> "Add External Archives"
3. Navigate to BroNexus/google-gson-2.2.4-release/google-gson-2.2.4
4. Include both gson-2.2.4 and gson-2.2.4-sources
5. Repeat step 2
6. Navigate to BroNexus/commons-io-2.4.bin/commons-io-2.4
7. Include both commons-io-2.4 and commons-io-2.4-sources
8. Go to "Project" at the top, select "Clean"
9. Program should run normally
